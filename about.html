About the project

The GeOrchestra project

Objectives

1 Scientific purpose

This project is funded by the labex G-EAU-THERMIE PROFONDE. Its goal is to develop the knowledge in the deep geothermal field by mixing academic and industrial competences and to study the structure and the function of the deep geothermal reservoir of the Upper Rhine Graben area. With the EGS pilot plant of Soultz-sous-Forets, more than 20 years of knowledge have been collected. The "Centre de données de géothermie profonde" (the Data center of deep geothermal energy) has been created in order to preserve and promote the data. GeOrchestra is an open source spatial data infrastructure (SDI) that will be used in order to develop the diffusion platform. This tool will be used as a data repository throughout the GeoNetwork software.

2 Data

The data implemented in the platform are any data that has been collected in the Upper Rhine Graben area that could give useful information for a geothermal study. That could be hydraulic, seismology, seismicity, geologic, geodesic or geophysics data
